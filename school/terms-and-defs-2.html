Terms and Definitions

Term Definition
Internet Service Provider (ISP) A company that provides internet services to customers
Network Interface Card (NIC) A hardware that helps computer to connect to a network either via LAN or WLAN
Uniform Resource Locator (URL) A location of the a specific website or a file
Web browser A software application that used to access the internet by retreiving a web page from an webserver via url and displays it on your desktop
Modem (aka Modulator-Demodulator) An ADC hardware component that allows a electronic devices including routers and switches, to connect to the internet
Client A computer connects to and uses the resources of a remote computer, or server
Server A computer where clients connects to to retrieve data
Packet A unit of data that can be sent across a network
Cookies Data that are stored on the clients computer when a website is visited
