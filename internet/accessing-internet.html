Accessing the Internet

Page about accessing the Internet.

Make sure you have a computer, a router, a modem, an internet connection provided by your ISP, and a web browser.
Role of a modem, router, web browser and ISP to gain access to the Internet:

Role of a Web browser is to render HTML on your screen, this is what you see on your web browser software.

Your computer then uses the NIC to send data to the router/modem via LAN cable(wired) or WLAN(wireless). Wired connections are faster.

This device will digital data(1s and 0s) sent from your computer to analog signals so that it can be sent over fibre

After the data is passed through the router/modem, it will first reach your ISP's server which allows access to the internet, routing internet traffic around the world, resolving domain names etc...

Finally the data reaches the target server via fibre optics.
